■動作確認済み環境について
・Windows10
(実行に.NET Framework4.5.2以上が必要)

■機能
クリップボード内文言の末尾の改行を除去するツールです。

■開発の背景
Windows版Evernoteにおいて、Evernote内に保存した文言をコピーして
他のアプリに貼り付けを行う際に末尾に改行が増えてしまう事があり、
その対処用ツールとして作成しました。


■導入方法
【単純起動】
①LizardTailCutter_V1.0.zipを解凍
②解凍後のディレクトリ内に含まれるLizardTailCutter.exeをダブルクリックすることで起動
(起動後、タスクメニュー内にアイコンが表示されます)

【Windowsの起動時に同時起動するように設定する】
①LizardTailCutter.zipを解凍
②解凍後のディレクトリ内に含まれるLizardTailCutter.exeのショートカットを
　以下のディレクトリに配置
　C:\Users\ユーザー名\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

■使用方法
【自動モードON時】
自動モードON時はクリップボードに新たな文言がコピーされた事を自動検知し、
クリップボードに存在する文言の末尾の改行を除去します。

【自動モードOFF時】
自動モードOFF時はキーボードのコントロールキー+シフトキー+Bキーの３つのキーを
同時に押下する事でクリップボードに存在する文言の末尾の改行を除去します。


【自動モードのON/OFF切り替えについて】
LizardTailCutter起動後、タスクメニュー内に表示されるアイコンを右クリックすることで
表示されるメニューにて、「自動」を左クリックする事で自動モードのON/OFF切り替えを
行う事ができます。
(自動の文字左側にチェックが付与されている場合はON、無い場合はOFF)


【終了】
LizardTailCutter起動後、タスクメニュー内に表示されるアイコンを右クリックすることで
表示されるメニューにて、「終了」を左クリックする事で終了します。
